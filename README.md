# AJK Consorcios - versión 3

## Para usar en Streamlit Cloud
1. Subí estos archivos a un repositorio de GitHub.
2. Entrá a https://streamlit.io/cloud
3. Elegí "New app"
4. Seleccioná tu repo y el archivo `app.py`
5. Deploy

## Archivos
- app.py
- requirements.txt
- ajk_logo.jpg

## Importante
La app guarda la información en `consorcios_data.json`.
Si la usás en Streamlit Cloud, para no perder datos lo ideal es una siguiente versión con base de datos (Supabase o Google Sheets).
